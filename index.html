Comment acheter une paire ?+

En ligne, en trois minutes : choisissez votre monture et votre coloris, puis suivez le mode de confirmation disponible. Vous pouvez aussi réserver un essayage privé, sans engagement.

Le paiement est-il sécurisé ?+

Chaque commande n'est confirmée qu'après vérification du paiement. Lorsque le paiement en ligne est disponible, il est traité par PayDunya ; sinon le concierge vous accompagne sur WhatsApp. Vous recevez une référence et un suivi privé.

Livrez-vous en dehors d'Abidjan ?+

Oui. La livraison est disponible dans le monde entier. Abidjan reste prioritaire ; pour les autres destinations, le concierge confirme le délai et les frais avant paiement.
